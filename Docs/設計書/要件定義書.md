## 1. プロジェクト概要

子供の貯蓄情報を家族で管理するWebアプリケーション。
複数の子供の入出金を記録し、残高・目標達成状況・貯蓄推移を可視化する。
パートナーや祖父母を招待して家族全員で成長を共有できる。

---

## 2. ユーザーとロール

### アカウント単位
- **親ごとに1アカウント**を作成する
- 1アカウントで複数の子供を管理する
- パートナー・祖父母を招待URLで追加できる

### ロール定義

| ロール | 対象 | できること |
|---|---|---|
| PARENT（親） | アカウント作成者 | 全操作（子供の登録・編集・削除、入出金記録、目標設定、家族招待） |
| PARTNER（パートナー） | 招待されたパートナー | 編集・管理可 または 閲覧のみ（招待時に設定） |
| GRANDPARENT（祖父母） | 招待された祖父母 | 閲覧のみ（デフォルト） |
| CHILD（子供） | 子供用ログインユーザー | 履歴の閲覧のみ（自分のデータのみ） |

## 3. 機能要件

### 3-1. ダッシュボード（家族全体）
- ナビゲーション：ダッシュボード / 家族 / （レポートは将来対応）
- 家族の純資産（全子供の合計残高）を大きく表示する
- 棒グラフで月次推移をヒーローカード内に表示する
- 今月の増減率（先月比）を表示する
- 「教育資金の目標」セクションで子供ごとのカードを表示する
  - 名前・年齢・目標名・残高・達成率プログレスバー・達成率%・目標金額
  - 「順調」「期待の星」などのバッジラベル
- 「貯蓄の分析」サイドパネルを表示する
  - 目標達成度ドーナツグラフ（%表示）
  - 今月の目標金額
  - 自動積立の有効/無効状態
- 「最近の履歴」セクション（全子供分の直近3〜5件）
- 「バッジ・実績」セクション：将来対応（スコープ外）
- ヘッダー右側：子供を追加ボタン・新規入力ボタン・ユーザーアイコン

### 3-2. 子供管理
- 子供を複数登録できる（人数上限なし）
- 各子供に名前・アバター画像/絵文字・テーマカラー・生年月日・学年を設定できる
- 子供の編集・削除ができる

### 3-3. 入金記録
- 以下のカテゴリで入金を記録できる

| カテゴリID | 表示名 | アイコン |
|---|---|---|
| allowance | おこづかい | payments |
| earned | お手伝い | volunteer_activism |
| new_year | お年玉 | redeem |
| other | その他 | more_horiz |

- 記録項目：金額・カテゴリ・日付・メモ（任意）

### 3-4. 出金記録
- 使ったお金を記録できる
- 記録項目：金額・日付・メモ（任意）

### 3-5. 残高表示
- 全入金合計 − 全出金合計 = 現在残高を表示する
- 先月比の増減率を表示する

### 3-6. 目標設定
- 子供ごとに目標を複数設定できる
- 設定項目：目標名・目標金額・目標達成日・貯蓄の目的カテゴリ・お子様へのメッセージ（任意）
- 達成率をプログレスバーで表示する
- 残高 ≥ 目標金額 になった時点で「達成バッジ」を表示する

### 3-7. 履歴表示
- 子供ごとに入出金履歴を一覧表示する
- 月ごとに絞り込みができる
- 各履歴に：日付・カテゴリアイコン・カテゴリ名・金額・口座名を表示する

### 3-8. 貯蓄推移グラフ
- 子供詳細画面に折れ線グラフを表示する（Chart.js）
- 期間切り替え：6ヶ月 / 1年

### 3-9. クイック操作（子供詳細）
- 貯金する / 引き出す / リマインダー / シェア の4ボタンを表示する
- 「貯金する」→ 入金登録画面へ
- 「引き出す」→ 出金登録画面へ
- リマインダー・シェアは将来対応

### 3-10. 家族情報連携
- パートナー・祖父母を招待URLで追加できる
- 招待時に共有権限（閲覧のみ / 編集・管理可）を設定できる
- 共有中の家族一覧・招待承認待ち・解除機能
- 最近の共有アクティビティを表示する

## 4. 非機能要件

### 4-1. 認証・セキュリティ

| 項目 | 内容 |
|---|---|
| ログイン方式 | メールアドレス＋パスワード **および** Googleログイン（OAuth2.0）の両方対応 |
| セッション管理 | Spring Securityのセッションベース認証 |
| パスワード保存 | BCryptでハッシュ化して保存（平文保存禁止） |
| アクセス制御 | ロールベース（PARENT / PARTNER / GRANDPARENT / CHILD）でURLレベルのアクセス制限 |
| CSRF対策 | Spring SecurityのCSRFトークンを有効化 |
| データ分離 | 親アカウントは自分の子供データのみ参照・操作可能 |

### 4-2. ロール別アクセス制御

| 機能 | PARENT | PARTNER（編集） | PARTNER（閲覧） | GRANDPARENT | CHILD |
|---|---|---|---|---|---|
| 子供の登録・編集・削除 | ✅ | ❌ | ❌ | ❌ | ❌ |
| 入出金の記録・編集・削除 | ✅ | ✅ | ❌ | ❌ | ❌ |
| 目標の設定・編集・削除 | ✅ | ✅ | ❌ | ❌ | ❌ |
| 履歴・残高・グラフ閲覧 | ✅ | ✅ | ✅ | ✅ | ✅（自分のみ） |
| 家族招待・管理 | ✅ | ❌ | ❌ | ❌ | ❌ |

### 4-3. UI・UX

| 項目 | 内容 |
|---|---|
| デザインシステム | DESIGN.md準拠（The Gentle Guardian） |
| ロゴ | 🐷豚の貯金箱アイコン（Material Symbols: savings）＋「こどちょ」テキスト |
| フォント | Plus Jakarta Sans（見出し）/ Manrope + Noto Sans JP（本文） |
| カラー | primary #0060ad / surface #f8f9ff / secondary #1a6b59 / tertiary-container #feb246 |
| レスポンシブ対応 | スマートフォン・PC両対応 |
| 対応ブラウザ | Chrome・Safari・Edge 最新版 |

### 4-4. インフラ・運用

| 項目 | 内容 |
